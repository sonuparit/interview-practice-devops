# Mid-level DevOps Practice for International Remote Jobs

This repository is a structured learning resource for aspiring and practicing DevOps Engineers preparing for mid-level DevOps interviews at international remote companies.

It covers the core concepts, tools, services, and real-world practices that are commonly expected during technical interviews. Each topic includes a dedicated README with focused explanations, practical knowledge, and interview-oriented content to help you build both theoretical understanding and hands-on skills.

Whether you're transitioning into DevOps or strengthening your existing knowledge, this repository is designed to provide a clear roadmap for interview preparation and production-ready learning.

## What You'll Learn

This repo is targeted for Mid-Level DevOps Interview practice for international market and you will learn almost everything which is important in:
- Linux
- Docker
- K8s
- Terraform
- GitHub Actions
- AWS
- Ansible
- Observability
- Python Scripting
- Bash Scripting
